当前已确认的五点描述偏好：

- 偏好强逻辑型五点，不要漂亮废话。每条尽量回答：为什么这个卖点重要、产品靠什么实现、对应哪些具体场景、有什么边界或注意事项。
- 五点标题承担 SEO 和扫读功能。每条 bullet 的 Header 一般 3-7 个单词，优先融合“核心品类词或品类词变体 + 买家结果/使用价值”；核心品类词或 TOP1 关键词优先考虑进入第 1 条 Header，但不能为了 SEO 牺牲自然度、准确性或风险控制。输出格式偏好为：Header 使用 Unicode 加粗字符，后接冒号和空格，再接普通正文。
- 偏好本地化场景，但必须先判断目标上架市场，再决定是否本地化和本地化强度。高气候/生活方式相关品类可以更明显结合当地使用环境；偏通用功能产品弱化本地化，优先讲兼容性、场景和参数；装饰、拼图、礼品类更适合结合当地审美、礼品场景或节日语境；带插头或电气参数的产品，可以把插头类型、电压、适配器、充电器规格等已确认合规信息作为本地化内容。不要假设一定是某个国家，也不要把不相关产品强行本地化。
- 偏好“参数 + 用户结果”绑定出现。关键参数后面要说明它带来的实际好处，例如容量减少补水次数、噪音不打扰睡眠、续航支持全天使用、距离适合会议室或教室。
- 可以使用“普通产品 vs 本产品”的隐性对比来强化转化，但只在白牌差异不明显、用户痛点明确、且产品事实支持时使用，不要滥用。
- Header 可以有画面感，尤其适合礼品、装饰、玩具和女性产品；正文必须回到事实、参数、配件、摆放场景或限制条件。
- 偏好把容易引发差评的限制条件自然写进第 4 或第 5 条，例如电压/插头、传输距离、兼容性、TF 卡或电池不包含、耗材、水质、适用人群等。
- 允许高端化表达低价白牌产品，例如 engineered、professional、premium、advanced、smart、precision、reinforced、stable、gift-ready、studio-quality、salon-grade，但必须有材料、结构或参数支撑。
- 偏好克制处理效果边界：能写 helps reduce / supports / designed for / suitable for 时，不写 prevent / guaranteed / completely / without disturbing others 这类容易过满的表达；漂亮比喻如果不能落回事实和用户结果，宁可不用。
