# Share

这里分享的是部分可复用的 Amazon Listing Prompt 资产。

- `prompts/`：任务级 Prompt 本体。
- `preferences/`：对应 Prompt 的偏好补充片段。

你可以基于这些分享内容和提示词，构建自己的 Listing 生成 Skill；也可以把其中的 Prompt 与偏好片段拆出来，辅助在其他 AI 工具中使用。
